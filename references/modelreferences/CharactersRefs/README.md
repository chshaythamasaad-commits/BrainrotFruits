# BrainrotFruits Reference Pack

Drop the `References` folder into the root of the Roblox/Codex repo.

Recommended repo path:
`BrainrotFruits/References/Characters/<CharacterName>/Base/<CharacterName>_Base_Turnaround.png`

The base sheets are the source of truth for the current voxel/chibi style. The earlier smooth toy renders should not be used as the main style unless intentionally making a different asset set.

Codex should use these references to create placeholder Roblox models, metadata, and variant definitions without trying to perfectly generate complex meshes automatically.
