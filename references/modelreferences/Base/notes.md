# BrainrotFruits Base Plot Reference Notes

Use the saved `base-reference` image as the main visual reference for the player plot model.

This reference defines the desired layout, proportions, and style for each player plot.

## Core Goal

Create a polished Roblox simulator-style player base/plot.

The plot should feel like a small colorful voxel island base where players display their collected Brainrot Fruits, earn money, and show off rare characters.

The plot should look polished, readable, blocky, and toy-like.

Do not make it a flat empty rectangle with glowing pads.

## Style Direction

- Colorful Roblox simulator style
- Voxel / block-built
- Bright and readable
- Cute tycoon/simulator base feeling
- Mobile-friendly
- Clean layout
- Low clutter
- Not realistic
- Not overly detailed
- Not copied from another game

Use the image for construction and layout reference only.
Do not copy external assets or another game's exact models.

## Plot Size

Recommended plot size:

- Width: around 70 studs
- Depth: around 55 studs
- Slightly raised above the surrounding ground

The plot should have:

- raised grass foundation
- stone/block edge border
- wooden fence
- front gate
- path to the central hub

## Required Plot Layout

The fruit platforms should go vertically, not horizontally.

There should be:

- one vertical set of fruit platforms on the left side
- one vertical set of fruit platforms on the right side
- catapult in the middle
- clear center lane/path
- hut at the back
- spawn pad near front-left
- collector/money machine near front-right
- owner sign at the front gate
