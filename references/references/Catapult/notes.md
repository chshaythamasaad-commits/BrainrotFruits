# BrainrotFruits Catapult Reference Notes

Use the saved catapult reference image as the main visual reference for the BrainrotFruits launcher model.

This catapult should be the main launcher identity of the game.

## Core Goal

Create a blocky wooden catapult that feels like it belongs in a colorful Roblox simulator game.

The catapult should be:

- readable from far away
- easy to understand as a launcher
- blocky / voxel / Roblox-built
- wooden with metal accents
- toy-like and playful
- not realistic or overly detailed
- mobile-friendly
- suitable for younger Roblox players

## Why Catapult

BrainrotFruits should use a catapult instead of a cannon as the main launcher because it feels:

- goofier
- more original
- more toy-like
- better for launching funny fruit characters
- more visually fitting for the voxel island/plot style

Cannons can be added later as launcher skins or upgrades, but the default identity should be catapult.

## Visual Style

Target style:

- chunky wooden beams
- simple block construction
- square/rectangular shapes
- clear throwing arm
- clear basket/cup at the end
- sturdy base frame
- metal brackets and bolts
- simple rope/hinge details
- no smooth realistic geometry
- no complex mesh requirement

Use mostly Roblox-generated parts:

- Blocks
- Wedges
- Cylinders only for hinge/axle parts if needed
- Small cubes for bolts
- Simple brown/wood colors
- Gray metal accents

## Required Catapult Parts

The model should include:

1. Base Frame
   - rectangular wooden base
   - chunky beams
   - stable footprint
   - sits flat on the ground or platform

2. Side Supports
   - two tall support towers
   - A-frame style supports
   - angled beams for strength
   - one support on each side

3. Throwing Arm
   - long wooden beam angled upward
   - pivots around a central axle
   - should clearly show launch direction

4. Basket / Cup
   - square or blocky cup at the top/end of the arm
   - big enough to visually hold a Brainrot Fruit
   - should be open on top

5. Hinge / Axle
   - horizontal crossbar through the support towers
   - gray metal rings/brackets
   - small bolt blocks

6. Front Stop / Brace
   - small metal or wood stopping bracket near the base
   - makes the catapult look functional

7. Decorative Details
   - metal plates
   - bolt cubes
   - rope-like brown/gray strips
   - small plank details

## Scale

For player plots:

- Catapult should fit in the center of the plot
- Should not block all movement
- Should be large enough to read clearly
- Suggested size:
  - Width: 10 to 14 studs
  - Length: 14 to 18 studs
  - Height: 8 to 12 studs

For shared central launch:

- Can be larger and more dramatic
- Suggested size:
  - Width: 16 to 22 studs
  - Length: 22 to 30 studs
  - Height: 14 to 20 studs

## Placement on Plot

When used inside a player plot:

- place catapult in the exact middle of the plot
- place fruit pads vertically on both sides
- keep a clear center path
- catapult should face toward the front or toward the launch direction
- keep it decorative unless gameplay requires otherwise

Recommended plot layout:

```txt
Back of Plot

[ Hut / Trees / Decorations ]

Left Fruit Pads        Right Fruit Pads
[Pad]                  [Pad]
[Pad]                  [Pad]
[Pad]    Catapult      [Pad]
[Pad]    Center        [Pad]
[Pad]                  [Pad]
[Pad]                  [Pad]

[Spawn]   [Gate/Sign]   [Collector]

Front Path to Hub
```
