# BrainrotFruits Center Area Reference Notes

Use the saved center-area reference image as the main visual reference for the central hub / main launch area.

This reference defines the desired layout, mood, and readability of the center of the map.

## Core Goal

Create a polished, colorful, kid-friendly central area where players immediately understand:

1. This is the main place to launch.
2. The catapult is the main attraction.
3. The launch lane goes forward from the catapult.
4. Player plots are around the outside.
5. Global leaderboards and showcase objects are in the hub.

The center should feel exciting, inviting, and addictive to walk around.

## Naming

Use:

- Main Launch
- Launch Zone
- Launch Lane

Do **not** call it:

- Shared Lunch
- Shared Launch
