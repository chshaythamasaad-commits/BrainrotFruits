# Diamond Strawberita

Base character:
- Strawberita

Audience:
- PG / Roblox-safe / child-friendly

Variant type:
- Legendary / shiny visual variant

Visual direction:
- strawberry body becomes glossy diamond/crystal red
- seeds become small bright crystal studs
- leaf crown becomes emerald-like
- add sparkle particles
- add reflective-looking highlights using simple Roblox parts/materials
- keep chibi mascot proportions
- no adult proportions
- no revealing outfit
- no suggestive pose

Gameplay direction:
- much higher income than base Strawberita
- should look expensive and flex-worthy
- idle animation: proud pose, sparkle burst, tiny bounce

Roblox implementation:
- part-based model
- use Glass, Neon, SmoothPlastic, and simple gem shapes
- readable from far away
- optimized for mobile
