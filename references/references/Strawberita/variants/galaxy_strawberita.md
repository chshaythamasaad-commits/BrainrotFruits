# Galaxy Strawberita

Base character:
- Strawberita

Audience:
- PG / Roblox-safe / child-friendly

Variant type:
- Mythic / premium visual variant

Use base Strawberita structure, but change the visual theme.

Visual direction:
- strawberry body becomes deep purple/red galaxy texture style
- glowing star freckles instead of normal strawberry seeds
- leaf crown becomes dark green with glowing blue/purple edges
- add small orbiting star particles
- add soft cosmic aura ring behind character
- keep chibi mascot proportions
- no adult proportions
- no revealing outfit
- no suggestive pose

Gameplay direction:
- higher income than base Strawberita
- should feel rare and magical
- idle animation: floats slightly, sparkles, then does a peace sign

Roblox implementation:
- part-based model
- use simple spheres, wedges, cylinders, decals, particles, and neon parts
- readable from far away
- optimized for mobile
