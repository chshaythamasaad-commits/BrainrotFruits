# BrainrotFruits Creator Store Mesh/Asset Candidates

Research date: 2026-05-13

Purpose: optional map and environment polish candidates for BrainrotFruits. These are not approved replacements for the game's original procedural systems yet. Strawberita must remain an original block-built character made for this game.

## Research Notes

- Source: Roblox Creator Store search and item details APIs, with Roblox Creator Store documentation as usage context.
- Preference: free Creator Store assets, verified creator when available, no embedded scripts, and a bright blocky/low-poly style that can support a simulator/tycoon map.
- Safety rule: before using any asset, insert it into a blank Studio file, inspect all descendants, remove any scripts unless intentionally needed, verify dependencies, and confirm it does not visually copy a specific game or outside IP.
- Recommendation meanings:
  - Use: good candidate for Studio review and possible map decoration.
  - Maybe: useful idea, but needs extra visual/script/IP review.
  - Avoid: do not use for this project.

Roblox documentation notes that the Creator Store contains assets made by Roblox and the community for use in experiences, but models can include dependencies and scripts, so asset inspection is still required before use.
