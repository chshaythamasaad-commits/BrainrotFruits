# BrainrotFruits Blocky Character Style References

Research date: 2026-05-13

Purpose: visual research for improving Strawberita's original block-built character factory. These Creator Store / Toolbox assets are style references only. Do not import them into the live game or use them as final Strawberita assets.

## Safety Notes

- Source: Roblox Creator Store search/item metadata and thumbnail review.
- Roblox models can contain scripts and dependencies, so every candidate must be inspected in a blank Studio place before any import into the project.
- Assets that reference specific games, franchises, or suspicious tag spam are marked `Avoid`.
- Strawberita remains original. The useful lessons here are about proportion, face readability, and block construction, not copying a character.

## Style Lessons Borrowed

- Use a flat, intentional face zone instead of facial details scattered across the body.
- Keep eyes large, symmetrical, high-contrast, and centered on the face plane.
- Use one simple highlight per eye. More eye detail starts to look uncanny at small scale.
- Keep the mouth tiny and pixel-like. A shallow blocky smile reads better than a detailed open mouth.
- Put accessories near the top or side so they do not cross the face.
- Use squat proportions: big block body, short limbs, chunky feet.
- Preserve the silhouette with a few readable large blocks instead of many tiny details.

## Design Translation For Strawberita

The research points to a clearer Strawberita recipe:

- Treat the strawberry body like a cute block collectible, not a fruit sculpture.
- Face panel should be a large clean tile on the front.
- Eyes should be square, same size, same height, and evenly spaced.
- Pupils should be centered and calm, with one small highlight.
- Mouth should be a tiny pixel smile, not a detailed open shape.
- Bow/accessory should move above the face, so it reads as decoration and not face clutter.
- Seeds should frame the face instead of competing with it.
